Project Demonstration

This page outlines the step-by-step process for implementing the VoIP call center project using FreePBX, Asterisk, and softphones.

Step 1: Create a Virtual Machine

After downloading the FreePBX ISO file on the website, we create a virtual machine using VirtualBox.

Step 2: Connect to the FreePBX Web Interface

Once FreePBX is installed, access its web-based GUI, using the IP address of the server.

Step 3: Create Extensions

Extensions are required for softphones and the IVR. Here's what we choose to configure them:

We create three SIP account to connect the three softphones and one virtual extension for the IVR. For these extension we need unique number associated with them, here we choosed 201, 202, 203 and 2000

Step 4: Configure the Queue and IVR

Set up the call routing and IVR to manage calls effectively:

In order to manage serveral calls at the same time, we need to make queues. One for each phone we will be able to call through the IVR, here it's 201 and 202.

We also need to make an IVR to manage all the incoming calls to the right service, with a voice to explain and a music while on hold.

Step 5: Connect Softphones to the Asterisk Server

Each softphone must be configured to connect to the FreePBX server:

To do so, we need the IP address of the server, the extension number and the password associated with the extension.

In this project, we used PhonerLite, Yane and MicroSIP as softphones.

Final Test

To test the system:

  • Call the IVR extension (2000) from an external from one of the softphones (201).
  • Use the IVR menu to reach the phone 202
  • Call the IVR again from a different phone (203) and reach the phone 202 again to verify the good functioning of the queue.
